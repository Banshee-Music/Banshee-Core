# Banshee BNB AI Agent

This agent is the marketplace operator.

## Jobs

1. Verify artists.
2. Review Greenfield submissions.
3. Create ticket listings.
4. Airdrop tickets.
5. Read SubQuery performance totals.
6. Mint BANSHEE rewards to artists.

## Install

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run

```bash
python music_agent.py
```
